# TokenCount

Token counter for texts, using GPT-compatible tokenization.

## About
This project allows you to calculate the number of tokens in a text, simulating the tokenization used by GPT models. The interface is modern, responsive, and easy to use.

## How to use
1. Type or paste your text in the box.
2. See the token count in real time.

## Technologies
- HTML, CSS, JavaScript
- [gpt-tokenizer](https://www.npmjs.com/package/gpt-tokenizer)

## Deploy
Recommended to publish via Vercel, integrating with this repository.

## Deploy on Vercel
To publish this project on Vercel:

1. Log in to https://vercel.com with your account (GitHub recommended).
2. Click "Add New... > Project" and select the `TokenCount` repository.
3. Follow the steps to import and publish.
4. Vercel automatically detects static projects. Just confirm and wait for the deploy.

After deploy, your site will be available at a URL generated by Vercel.
