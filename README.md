# TokenCount

Contador de tokens para textos, usando tokenização compatível com GPT.

## Sobre
Este projeto permite calcular o número de tokens de um texto, simulando a tokenização utilizada por modelos GPT. A interface é moderna, responsiva e fácil de usar.

## Como usar
1. Digite ou cole seu texto na caixa.
2. Veja a contagem de tokens em tempo real.

## Tecnologias
- HTML, CSS, JavaScript
- [gpt-tokenizer](https://www.npmjs.com/package/gpt-tokenizer)

## Deploy
